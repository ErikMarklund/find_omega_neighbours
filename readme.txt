FInds protein assemblies in the pdb that best matches a given point in the mass-omega plane. Requires python2.7

USAGE:   python find_omega_neighbours.py -mass=<float>  (-ccs=<float> | -omega=<float>) [ -db=name_of_pdb_database ] [ -rank=[<int> | all] ] [ -massweight=<float> ] [ -omegaweight=<float> ] [ -h ]
